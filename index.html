What this implements

  • One-shot precompute of a similarity image.
  • Real-time mouse tracking with crosshair and diagonal trace.
  • Window, threshold, zoom, and fit controls to mimic Dotter-style exploration.
  • Drag-and-drop FASTA loading plus PNG/SVG export.
  • Pure static files, so it can be published directly via GitHub Pages.

Notes

This version uses a lightweight windowed similarity score rather than the full historical Dotter algorithm, but the interaction model matches the “precompute once, inspect instantly” workflow you described.
